## How to Use

Every piece of significant knowledge gets filed here.

When an agent makes a decision → log it.  
When a bug is found → log it.  
When a system is designed → document it.  
When research is completed → file it.

The goal: any future session of Cap DevRoom can open this folder and understand the entire system without needing a human explanation.
