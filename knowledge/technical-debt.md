## Protocol

New debt is logged here whenever:
- A known shortcut is made in engineering
- A refactor is deferred to ship faster
- A legacy pattern is identified in a codebase review
- The CTO Agent flags a structural issue

Each item must have an owner and a target resolution sprint.
